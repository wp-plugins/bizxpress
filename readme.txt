== Installation ==

There are 3 ways to install bizXpress:

= 1. From Your WordPress Admin =
1. In your Admin, go to menu Plugins > Add New
1. Search for 'bizXpress'
1. Click to install
1. Activate the plugin
1. A new menu 'bizXpress' will appear in your Admin

= 2. Download the Plugin =
1. Download the plugin (.zip file) on the right column of this page
1. In your Admin, go to menu Plugins > Add New
1. Select the tab "Upload"
1. Upload the .zip file you just downloaded
1. Activate the plugin
1. A new menu 'bizXpress' will appear in your Admin

= 3. Via FTP =
1. Download and unzip 'bizXpress.zip'
1. Upload the unzipped 'bizXpress' folder to the '/wp-content/plugins/' directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. A new menu 'bizXpress' will appear in your Admin
